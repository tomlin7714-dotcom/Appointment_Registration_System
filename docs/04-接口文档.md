# 医疗预约挂号系统 —— API 接口文档

> 基础URL：`http://localhost:8083`
> 请求头：`Content-Type: application/json`

---

## 1. 用户端接口

### 1.1 用户注册

```
POST /api/user/register
```

**参数：**
```json
{
  "phone": "13800138000",
  "pwd": "123456",
  "realName": "张三",
  "idCard": "110101199001011234"
}
```

**返回值：**
```json
{"code": 200, "msg": "操作成功", "data": null}
```

### 1.2 用户登录

```
POST /api/user/login
```

**参数：** `{"phone": "13800138000", "pwd": "123456"}`

**返回值：**
```json
{
  "code": 200,
  "data": {
    "userId": 1,
    "token": "user_token_1",
    "userInfo": { "id": 1, "phone": "13800138000", "realName": "张三", ... }
  }
}
```

### 1.3 获取用户信息

```
GET /api/user/info?userId=1
```

**返回值：** `{"code": 200, "data": {"id": 1, "phone": "13800138000", "realName": "张三", ...}}`

### 1.4 更新用户信息

```
POST /api/user/info/update
```

**参数：** `{"id": 1, "realName": "张三", "idCard": "110101199001011234", "avatar": "http://..."}`

### 1.5 头像上传

```
POST /api/user/avatar/upload
```

**参数：** `file` (MultipartFile)

**返回值：** `{"code": 200, "data": {"avatarUrl": "/uploads/avatars/xxx.jpg"}}`

### 1.6 修改密码

```
POST /api/user/pwd/modify
```

**参数：** `{"phone": "13800138000", "oldPwd": "123456", "newPwd": "654321"}`

### 1.7 创建预约

```
POST /api/user/appointment/create
```

**参数：**
```json
{
  "numberSourceId": 1,
  "userId": 1,
  "patientName": "张三",
  "patientIdCard": "110101199001011234"
}
```

**返回值：** `{"code": 200, "data": {"appointmentId": 1, "fee": 0.01}}`

### 1.8 支付

```
POST /api/user/appointment/pay
```

**参数：** `{"appointmentId": 1, "payType": "wechat"}`

### 1.9 预约列表

```
GET /api/user/appointment/list?userId=1&status=1
```

**参数：** `userId`(必填), `status`(可选: 0待支付 1待就诊 2就诊中 3已完成 4已取消)

### 1.10 预约详情

```
GET /api/user/appointment/detail?appointmentId=1
```

### 1.11 取消预约

```
POST /api/user/appointment/cancel
```

**参数：** `{"appointmentId": 1}`

### 1.12 预约统计

```
GET /api/user/appointment/stats?userId=1
```

**返回值：** `{"code": 200, "data": {"pending": 2, "completed": 3, "total": 5}}`

### 1.13 保存问诊单

```
POST /api/user/consultation/save
```

**参数：**
```json
{
  "appointmentId": 1,
  "formType": "初诊",
  "chiefComplaint": "头痛发热3天",
  "presentIllness": "现病史描述",
  "pastHistory": "既往史描述"
}
```

### 1.14 问诊详情

```
GET /api/user/consultation/detail?appointmentId=1
```

### 1.15 问诊历史列表

```
GET /api/user/consultation/list?userId=1&keyword=头痛
```

### 1.16 科室列表

```
GET /api/user/dept/list
```

### 1.17 医生列表

```
GET /api/user/doctor/list?deptId=1&keyword=张
```

### 1.18 排班列表

```
GET /api/user/schedule/list?doctorId=1&date=2026-07-03
```

### 1.19 号源列表

```
GET /api/user/schedule/list?doctorId=1&date=2026-07-03
```

### 1.20 AI导诊

```
POST /api/ai/triage
```

**参数：**
```json
{"symptom": "头疼发热"}
```

**返回值：**
```json
{
  "code": 200,
  "data": {
    "advice": "根据您描述的症状，建议优先考虑以下科室...",
    "depts": [
      {"deptId": 1, "deptName": "内科"},
      {"deptId": 12, "deptName": "呼吸内科"}
    ]
  }
}
```

## 2. 医生端接口

### 2.1 医生登录

```
POST /api/doctor/login
```

**参数：** `{"doctorAccount": "D001", "pwd": "123456"}`

### 2.2 排班保存

```
POST /api/doctor/schedule/save
```

**参数：** `{"doctorId": 1, "date": "2026-07-03", "period": "morning"}`
> period: morning/afternoon/evening

### 2.3 自动排班

```
POST /api/doctor/schedule/auto
```

**参数：** `{"doctorId": 1}`

### 2.4 排班列表

```
GET /api/doctor/schedule/list?doctorId=1
```

### 2.5 清理过期排班

```
GET /api/doctor/schedule/cleanExpired?doctorId=1
```

### 2.6 发布号源

```
POST /api/doctor/numbersource/save
```

**参数：** `{"scheduleId": 1, "totalNum": 20, "fee": 0}`

### 2.7 号源列表

```
GET /api/doctor/numbersource/list?doctorId=1&page=1&pageSize=9&date=2026-07-03
```

### 2.8 关闭号源

```
GET /api/doctor/numbersource/close?id=1
```

### 2.9 删除号源

```
GET /api/doctor/numbersource/delete?id=1
```

### 2.10 今日预约

```
GET /api/doctor/appointment/today?doctorId=1
```

### 2.11 预约列表（带筛选）

```
GET /api/doctor/appointment/list?doctorId=1&page=1&pageSize=9&keyword=张&appointmentId=1
```

### 2.12 开始就诊

```
POST /api/doctor/call/start
```

**参数：** `{"appointmentId": 1}`

### 2.13 完成就诊

```
POST /api/doctor/appointment/complete
```

**参数：** `{"appointmentId": 1}`

### 2.14 下一个叫号

```
GET /api/doctor/call/next?doctorId=1
```

### 2.15 工作台数据

```
GET /api/doctor/work/dashboard?doctorId=1
```

### 2.16 医生统计

```
GET /api/doctor/stats?doctorId=1
```

### 2.17 医生信息

```
GET /api/doctor/info?doctorId=1
```

### 2.18 获取问诊单

```
GET /api/doctor/consultation/get?appointmentId=1
```

### 2.19 保存诊断

```
POST /api/doctor/consultation/save
```

**参数：**
```json
{
  "appointmentId": 1,
  "diagnosis": "上呼吸道感染",
  "treatment": "口服抗生素3天",
  "examination": "体温38.5°C，咽部充血"
}
```
> 保存后自动更新预约状态为已完成

### 2.20 更新预约

```
POST /api/doctor/appointment/update
```

**参数：** `{"id": 1, "patientName": "张三", "patientIdCard": "...", "status": 1}`

### 2.21 取消预约

```
POST /api/doctor/appointment/cancel
```

**参数：** `{"appointmentId": 1}`

### 2.22 个人信息

```
GET /api/doctor/profile/get?doctorId=1
POST /api/doctor/profile/update
POST /api/doctor/profile/changePwd
```

## 3. 管理员端接口

### 3.1 登录

```
POST /api/admin/login
```

**参数：** `{"adminAccount": "admin", "pwd": "admin123"}`

### 3.2 系统概览

```
GET /api/admin/stats/overview
```

**返回值：**
```json
{
  "code": 200,
  "data": {
    "deptCount": 5,
    "doctorCount": 3,
    "userCount": 10,
    "appointmentCount": 8,
    "totalAppointments": 50,
    "scheduleCount": 120,
    "numberSourceCount": 480
  }
}
```

### 3.3 科室管理

```
GET  /api/admin/dept/list?page=1&pageSize=10
POST /api/admin/dept/save     {"name": "内科", "remark": "...", "area": "1区", "roomNumber": "101"}
POST /api/admin/dept/modify   {"id": 1, "name": "内科", ...}
GET  /api/admin/dept/get?id=1
GET  /api/admin/dept/del?id=1
```

### 3.4 医生管理

```
GET   /api/admin/doctor/list?keyword=张&deptId=1&status=0&page=1&pageSize=10
POST  /api/admin/doctor/save    {"name": "张医生", "doctorAccount": "D001", ...}
POST  /api/admin/doctor/modify  {"id": 1, ...}
GET   /api/admin/doctor/get?id=1
GET   /api/admin/doctor/del?id=1
POST  /api/admin/doctor/autoSchedule {"doctorId": 1}
```

### 3.5 用户管理

```
GET   /api/admin/user/list?keyword=张&phone=138&startTime=2026-01-01&endTime=2026-07-03&status=0&page=1&pageSize=5
GET   /api/admin/user/get?id=1
POST  /api/admin/user/save    {"name": "张三", "phone": "138...", "pwd": "123456", "idCard": "..."}
POST  /api/admin/user/modify  {"id": 1, "name": "张三", "pwd": "", "idCard": "...", "status": 0}
GET   /api/admin/user/del?id=1
GET   /api/admin/user/status?id=1&status=1
```

### 3.6 问诊记录管理

```
GET   /api/admin/consultation/list?doctorName=张&startTime=2026-01-01&endTime=2026-07-03&page=1&pageSize=9
GET   /api/admin/consultation/get?id=1
POST  /api/admin/consultation/update  {"id": 1, "diagnosis": "...", "treatment": "...", "examination": "..."}
POST  /api/admin/consultation/status  {"id": 1, "status": 1}
GET   /api/admin/consultation/del?id=1
```

### 3.7 系统日志

```
GET /api/admin/log/list
```

### 3.8 系统参数

```
GET  /api/admin/sysParam/list
PUT  /api/admin/sysParam/update
```

## 4. 全局返回值格式

### 成功
```json
{"code": 200, "msg": "操作成功", "data": {...}}
```

### 失败
```json
{"code": 400, "msg": "参数错误", "data": null}
{"code": 401, "msg": "未授权", "data": null}
{"code": 404, "msg": "资源不存在", "data": null}
{"code": 500, "msg": "服务器内部错误", "data": null}
```

### 分页返回值
```json
{
  "code": 200,
  "data": {
    "list": [...],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```
