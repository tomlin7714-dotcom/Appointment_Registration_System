## 如何执行SQL脚本

### 方法1：在IDEA中执行（推荐）

1. 在IDEA中找到 `database.sql` 文件
   - 位置：`c:\java\Appointment_Registration_System\database.sql`

2. 右键点击 `database.sql` 文件

3. 选择以下选项之一：
   - "Run SQL Script"（运行SQL脚本）
   - "Execute SQL Script"（执行SQL脚本）
   - 或者在Database工具窗口中打开该文件

4. 确认执行，等待脚本完成

### 方法2：使用MySQL命令行

1. 打开命令提示符或PowerShell

2. 执行以下命令：
```bash
mysql -u root -p hospital_registration_system < c:\java\Appointment_Registration_System\database.sql
```

3. 输入密码：`Lintel7714.`

### 方法3：使用MySQL客户端工具

2. 连接到数据库（localhost，用户root，密码Lintel7714.）
